Guía de configuración de afiliados
6 comentarios numerados en el código. Ctrl+F cada número y sustituye el href con tu link de afiliado.
01
Regístrate en los programas de afiliados
MagicSchool (magicschool.ai/affiliate), Brisk Teaching (briskteaching.com), Curipod (curipod.com), Canva (canva.com/affiliates), Notion (notion.so/affiliates), MailerLite (mailerlite.com/referral).
02
Localiza los comentarios en el código
Busca "AFILIADO 1", "AFILIADO 2"... hasta el 6. Cada comentario está justo encima del botón correspondiente.
Ctrl+F → "AFILIADO 1" → reemplaza href="URL"
03
Sube el archivo a GitHub y despliega
git add partners_profesoria.html && git commit -m "afiliados configurados" && git push
